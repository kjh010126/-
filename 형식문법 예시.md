# 형식문법 예시
문법 G = ({S,A,B}, {a,b}, S, P)

𝑆 → 𝐴 // (시작기호 S가 A로 변환되면서 형성규칙의 시작을 알림)

𝐴 → 𝑎𝐵 ∣ 𝜆 // (𝐴는 𝑎𝐵 또는 공백(빈 문자열, 𝜆)으로 변환될 수 있음)

𝐵 → 𝐴𝑏 // (𝐵는 𝐴𝑏로 변환됨)


해석:

S → A (첫 번째 규칙)

𝐴 → 𝑎𝐵 (두 번째 규칙의 첫 번째 선택)

𝐵 → 𝐴𝑏 (세 번째 규칙)

𝐴 → 𝜆 (두 번째 규칙의 두 번째 선택)

이렇게 하면 aAb라는 문자열이 생성되고 𝐴가 𝜆로 대체되면 최종적으로 

𝑎𝑏라는 단말기호로 구성된 문자열이 생성됩니다.
